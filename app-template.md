# 摘要
产品介绍摘要和技术摘要。用简短的话术了解该产品是做什么的。

# 动机
为什么生态需要这样的项目，所解决的问题是什么？如果动机明显，则可以忽略该部分。

# 原理及具体可实行性描述
技术实现细节、如何实现该功能或该功能所需的技术点和技术安排（尽可能详尽）。

# 参考实现
某些技术细节点有无已有的解决方案，该部分是对于可行性的补充。

# 所需资源（可选）
所需要的研发资源，项目预算等。
